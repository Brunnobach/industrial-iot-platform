Fluxo de dados

Pipeline de telemetria industrial: sensores publicam via MQTT, a ingestão grava no InfluxDB e a API FastAPI alimenta este painel e o Grafana.

Sensores → MQTT (Mosquitto) → Ingestão → InfluxDB → FastAPI → Painel / Grafana

Componentes

  • FastAPI Consulta e alertas
  • InfluxDB 2.7 Séries temporais
  • Mosquitto 2.0 Broker MQTT + WebSockets
  • Docker Compose Stack local com healthchecks

Execução local

git clone https://github.com/Brunnobach/industrial-iot-platform.git cd industrial-iot-platform docker compose up -d

Com a stack no ar, use o modo Gateway Real para ler o broker local e a API em localhost:8000.
