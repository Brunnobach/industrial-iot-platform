📡 Industrial IoT Platform

Industrial IoT data platform for sensor monitoring, analytics and alerting — designed for biogas plants, manufacturing and energy operations.

View on GitHub

What this project demonstrates

MQTT InfluxDB FastAPI Grafana Docker Time-Series

A complete end-to-end IoT system: simulated sensors publish data via MQTT, a time-series database stores it, a FastAPI service provides queries and alerts, and Grafana visualizes everything.

Architecture

Sensors (simulated) → MQTT Broker → InfluxDB → FastAPI → Grafana Dashboard + Alerts

Quick Start

git clone https://github.com/Brunnobach/industrial-iot-platform.git
cd industrial-iot-platform
docker-compose up -d

Then access Grafana at http://localhost:3000 and API docs at http://localhost:8000/docs.
