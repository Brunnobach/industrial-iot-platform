Como os dados chegam aqui

Os sensores publicam no Mosquitto. Um serviço de ingestão grava no InfluxDB; a API FastAPI serve este painel e o Grafana.

Sensores → MQTT (Mosquitto) → Ingestão → InfluxDB → FastAPI → Painel / Grafana

O que sobe com o Compose

  • FastAPI Consultas e avaliação de alertas
  • InfluxDB 2.7 Séries temporais
  • Mosquitto 2.0 Broker MQTT com WebSockets
  • Docker Compose Stack local com healthchecks

Subir localmente

git clone https://github.com/Brunnobach/industrial-iot-platform.git cd industrial-iot-platform docker compose up -d

Com a stack no ar, escolha Gateway real para ler o broker e a API em localhost:8000.
