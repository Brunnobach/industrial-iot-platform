Estrutura de Componentes

Esta plataforma demonstra uma arquitetura robusta de ponta a ponta para captura de telemetria industrial:

Sensores (Simulação) ─▶ MQTT Broker (Mosquitto) ─▶ Ingestão Python ─▶ InfluxDB ─▶ API FastAPI ─▶ Painel / Grafana

Stack Tecnológica

FastAPI / Python 3.11 API de Baixa Latência
InfluxDB 2.7 Banco de Dados Temporal
Eclipse Mosquitto 2.0 MQTT Broker Websockets
Docker Compose Orquestração com Healthchecks

Início Rápido Local

Para executar a infraestrutura física completa localmente em sua máquina:

git clone https://github.com/Brunnobach/industrial-iot-platform.git cd industrial-iot-platform docker-compose up -d

O painel buscará os dados diretamente do contêiner e escutará no broker local no modo Gateway Real.
